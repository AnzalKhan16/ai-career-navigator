# AI Career Navigator

An AI-powered career guidance platform that helps users discover suitable career paths based on their interests, skills, and goals through intelligent recommendations.

---

## Features

- AI-powered career recommendations
- Skill-based career analysis
- Interactive dashboard
- Responsive modern UI
- Personalized career guidance
- Full-stack application architecture

---

## Tech Stack

### Frontend
- React.js
- Vite
- JavaScript
- CSS3

### Backend
- Node.js
- Express.js

### Deployment
- Vercel
- Render

### APIs
- AI Integration APIs

## Installation

### Clone Repository

```bash
git clone https://github.com/AnzalKhan16/ai-career-navigator.git

cd ai-career-navigator
```

---

### Frontend Setup

```bash
cd client

npm install

npm run dev
```

---

### Backend Setup

```bash
cd server

npm install

npm start
```
